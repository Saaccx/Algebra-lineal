# Álgebra Lineal I.
El repositorio contiene notas del curso de álgebra lineal I, estas notas se presentan tanto en pdf como en un archivo zip que contiene un archivo .tex junto con las imagenes correspondientes si se requiere editar.

### Sobre el material.
* **Origen:** El contenido se basa en su mayoría en las notas impartidas por el profesor Ivan M. R quien estuvo a cargo del curso.
* **Modificaciones:** Se reescribieron y añadieron algunos teoremas y se modificaron pruebas, con el objetivo de explicar los pasos de manera más simplificada.
* **Complemento Final:** Para complementar la sección 4 y 5 se utilizo el libro correspondiente en la bibliografía, que es el libro que el profesor también ocupa para sus notas.

## Prerrequisitos.
El material está diseñado asumiendo que el alumno ya cuenta con nociones de **Teoría de Ecuaciones** o conocimientos de **Números complejo y Matrices**.
